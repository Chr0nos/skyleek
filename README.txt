Toutes les fonctions suivent le format de documentation suivant:
	http://leekwars.com/forum/category-7/topic-6965

a savoir:
/**

*/

pour le bloc, et en parametres:
	@level	: le niveau requis pour la fonction
	@return : le retour de la fonction (le type)
	@ops	: nombre d'operations de la fonction (peu etre variable)
	le texte principal sera aussi pris en compte
